# お問い合わせメール設定

お問い合わせフォームは `march25kk@gmail.com` 宛てに送信します。送信元にも同じGmailアカウントを使用します。

## 必要な設定

1. `march25kk@gmail.com` のGoogleアカウントで2段階認証を有効にする。
2. Googleアカウントで16桁のアプリパスワードを発行する。
3. デプロイ先の環境変数 `GMAIL_APP_PASSWORD` に、発行したアプリパスワードを設定する。
4. サーバーを再起動または再デプロイする。

通常のGoogleアカウントのパスワードは使用せず、リポジトリにも保存しないでください。

別のGmailアカウントを送信元として使用する場合は、環境変数 `GMAIL_USER` も設定します。受信先は常に `march25kk@gmail.com` です。
